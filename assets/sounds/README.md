# 音效檔案說明

## 📁 需要的音效檔案

請在此目錄下放置以下 MP3 音效檔案：

1. `notification.mp3` - 預設通知音（建議 1-2 秒）
2. `alarm.mp3` - 鬧鐘聲（建議響亮、2-3 秒）
3. `ringtone.mp3` - 來電鈴聲（建議 3-5 秒）
4. `message.mp3` - 訊息提示音（建議簡短、0.5-1 秒）
5. `beep.mp3` - 嗶嗶聲（建議簡短、0.3-0.5 秒）

## 🎵 音效來源建議

### 免費音效網站
- [Freesound.org](https://freesound.org/) - 需註冊，大量免費音效
- [Zapsplat.com](https://www.zapsplat.com/) - 免費下載，品質良好
- [Mixkit.co](https://mixkit.co/free-sound-effects/) - 完全免費，無需註冊

### 使用系統音效
也可以從手機系統中提取音效：
- Android: `/system/media/audio/notifications/`
- iOS: 需要從其他來源獲取

## 🔧 臨時解決方案

如果暫時沒有音效檔案，APP 會自動使用系統預設提示音作為備援，不會崩潰。

## ⚙️ 技術規格

- **格式**: MP3（推薦）或 WAV
- **採樣率**: 44.1kHz 或 48kHz
- **位元率**: 128kbps 或更高
- **聲道**: 單聲道或立體聲均可
- **音量**: 建議正規化到 -3dB 至 0dB

## 📝 版權注意

請確保使用的音效檔案：
- ✅ 是您自己創作的
- ✅ 來自免費授權（CC0、Public Domain 等）
- ✅ 已購買商業授權
- ❌ 不要使用有版權保護的音效
